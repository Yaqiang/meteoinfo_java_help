Meteorological Data

Following meteorological data were supported at present:

l         NetCDF grid data

l         GRIB edition 1 and 2 data

l         GrADS binary grid and station data

l         HDF grid and swath data

l         ARL packed meteorological data

l         HYSPLIT model output data: concentration, particle and trajectory

l         Lon/Lat Station ASCII data

l         MICAPS 1, 3, 4, 7, 11 and 13 format data

l         ESRI ASCII Grid data

l         Surfer ASCII Grid data

l         MM5 output and intermediate data
