Introduction

MeteoInfo is freely available software designed to view and analyze meteorological and spatial data interactively. Some GIS functions were developed from ground level. There are two kinds of versions of MeteoInfo developed using C# and Java. This Java version of MeteoInfo is cross platform application and requires Java 6 or better be installed. See the Java.com website for a free download of Java if you do not have it already installed. MeteoInfo can also be run automatically using MeteoInfo scripting with the Jython or Groovy language. The main functions are packed in the MeteoInfo class library, which could be used to conveniently develop the software. This document is based on the version 1.0. Any suggestion is welcome.
